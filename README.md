# Social-Media-Analytics-Dashboard
# 📊 Social Media Analytics Dashboard | Power BI

## 📌 Project Overview

The **Social Media Analytics Dashboard** is an interactive Business Intelligence project developed using **Microsoft Power BI**. It provides a comprehensive analysis of social media engagement by visualizing key performance indicators (KPIs), audience demographics, user interests, professions, and geographical distribution.

The dashboard enables users to explore data through dynamic filters, making it easier to identify trends, measure engagement, and gain actionable insights for data-driven decision-making.

---

## 🚀 Features

* Interactive KPI Cards displaying:

  * Total Likes
  * Total Shares
  * Total Comments
  * Total 3-Second Video Views
  * Total 1-Minute Video Views

* Dynamic Slicers

  * Gender
  * Hobby
  * Profession
  * Country

* Interactive Visualizations

  * Likes, Shares, and Comments by Hobby
  * Video Views by Profession
  * Likes Distribution by Gender
  * Age Distribution of Users
  * Country-wise Share Distribution (Map)

* Cross-filtering across all visuals

* User-friendly and responsive dashboard design

---

## 📊 Dashboard Analysis

### 1. Overall Performance Dashboard

Provides a complete overview of social media engagement through KPI cards, interactive charts, and maps.

### 2. Hobby Analysis

Analyzes user engagement based on different hobbies such as Traveling, Gaming, Music, Dancing, Sports, Cycling, Reading, Photography, Painting, and Cooking.

### 3. Profession Analysis

Compares video views and engagement among various professions including Doctors, Developers, Scientists, Designers, Teachers, Entrepreneurs, Musicians, Artists, Writers, and Engineers.

### 4. Gender Analysis

Shows the percentage distribution of likes among Female, Male, and Non-binary users using a donut chart.

### 5. Age Distribution

Displays the distribution of users across different age groups segmented by gender.

### 6. Country Analysis

Visualizes total shares across different countries using an interactive world map, helping identify regional engagement trends.

### 7. Dynamic Filtering

Users can combine multiple filters (Gender, Hobby, Profession, and Country) to perform customized analysis and instantly update all dashboard visuals.

---

## 📈 Key Insights

* Female users generate the highest percentage of total likes.
* Traveling and Gaming are among the highest-engagement hobbies.
* Doctors and Developers receive the highest video views.
* Audience engagement varies significantly across professions.
* Geographic analysis highlights country-wise social media performance.
* Dynamic filters provide detailed demographic and behavioral insights.
